# Pro Components Mockups

These SVG mockups show the intended structure and component boundaries for the first version of `@rap/components-pro`.

- `01-pro-data-grid-top.svg`: list page with top `ProSearchBar`, toolbar, keyword search, and `ProDataGrid`.
- `02-left-search-split-layout.svg`: left search + split layout + grid.
- `03-page-overlay-form.svg`: `PageOverlay` for complex create/edit flows.
- `04-chunk-upload.svg`: chunk upload with MD5 worker and separate hash/upload progress.
